# LibManager

A package-manager to install libraries, currently only for Java will get ported to other languages

## What can it do?

Currently, it can only download libraries from this repository.

## How can i use this?

1. Go to the [Releases page](https://github.com/IkeVoodoo/LibManager/releases/latest) and download the program.
2. Either use it as a standalone program from the command line, or use it as a library. (documentation coming soon)
3. Enjoy
